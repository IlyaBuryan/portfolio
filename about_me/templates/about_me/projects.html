This site is made at an advanced level in the process of my learning the framework. It demonstrates ability to create a fully functional online store website.

In addition to all the standard Django toolkit mastered via the first site: working with forms, templates, views, models, urls, statics and media files, this project had the following features:

  • Complicated structure, consisting of 4 applications: own admin panel, app for basket and users.
  • Implemented complex queries to the database. Trained much in Shell.
  • Inheritance and redefinition of the class methods are applied when working with forms and models
  • Adminapp is totally performed via CBV
  • Used both get and post data transfer to the controllers
  • Pagination is implemented on product pages
  • Implemented user authentication and registration + page access decorators
  • Trained creating template filters and management scripts

Used stack of technologies:

  • Django Framework v3.1.6
  • Pure Python
  • HTML5, CSS3 (SCSS, Bootstrap)
  • JavaScript (+JQuery, Ajax)
  • Postgresql deploy
  • Support: Gimp, Figma, Validator, Typograf, Browsershots
